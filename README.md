# Breath VOC Dataset
﻿
This repository contains the breath volatile organic compounds (VOCs) concentration data used in this study, including both animal and human datasets.
﻿
## 1. Mouse breath VOC dataset
﻿
**File name:** DSS-EEN-HC_breath VOCs.xlsx  
﻿
- This dataset contains raw breath VOC concentration data from mouse experiments.
- A total of **33 VOCs** are included.
﻿
### Data structure:
- **Sheet 1 (NE group):**  
  Normal exhaled breath samples without feces or urine interference.
﻿
- **Sheet 2 (WE group):**  
  Breath samples collected with potential interference from feces and urine.
﻿
---
﻿
## 2. Human breath VOC dataset
﻿
**File name:** IBD-HC_breath VOCs.xlsx  
﻿
- This dataset contains breath VOC concentration data from human subjects.
- A total of **71 VOCs** are included.
- Sample size:
  - **IBD group:** 118 samples  
  - **Healthy controls (HC):** 118 samples  
﻿
---
﻿
## 3. Data format
﻿
- Rows represent individual samples  
- Columns represent VOC concentrations  
- All concentrations are reported in **ng/L**
﻿
---
﻿
## 4. Data availability and ethics
﻿
- All data have been **fully anonymized**.
- No personally identifiable information is included.
- Data are shared in accordance with ethical approval and relevant regulations.
